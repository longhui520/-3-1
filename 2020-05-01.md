- `js` **json和对象有什么区别？**
    - 有区别
    - json 是来描述和传输数据的一种数据格式
    - 对象则包含了对象原型
    - json通常代表json字符串，为了方便操作和获取json里的数据 可以转化为对象，对象也可以转化为json字符串
    - json 中包含 字符串，数字，布尔，数组和普通对象 等有限的数据类型
